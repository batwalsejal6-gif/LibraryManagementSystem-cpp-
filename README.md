# Library Management System in C++

A simple console-based Library Management System developed in C++ using arrays, functions, and basic file structure concepts.

This project allows admins and students to manage books, student accounts, balances, and book issuing operations.

---

# Features

## Admin Features
- Add new books
- Edit existing book details
- View all books and availability status
- View enrolled students
- View student balances

## Student Features
- Create account
- View balance
- Deposit money
- Issue books

---

# Technologies Used

- C++
- Arrays
- Functions
- Loops & Conditions
- String handling (`cstring`)
- Formatting (`iomanip`)
